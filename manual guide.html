Guide Manual Tool RPGM

Panduan translate game RPG Maker (Android)

Step 1 – Cari Game yang Ingin Ditranslate

Buka file manager lalu cari file game RPG Maker yang ingin diterjemahkan (.apk / .zip / .rar / folder game).

[Gambar CDN: Cari file game RPGM]

Step 2 – Ekstrak ke Folder Tujuan

Ekstrak file game hingga menjadi folder. Tool tidak bisa membaca file arsip langsung.

[Gambar CDN: Menu ekstrak file]

Step 3 – Masuk ke Folder Data Game

Buka folder hasil ekstrak lalu masuk ke: 

folder_game / www / data

Folder data berisi file JSON yang akan diterjemahkan.

[Gambar CDN: Struktur folder www/data]

Step 4 – Salin File JSON ke Folder Kerja

  • Cari file CommonEvents.json
  • Cari file MapXXX.json
  • Select All → Copy
  • Tempelkan ke folder baru (folder kerja)
Jangan cut. COPY saja agar file asli tetap aman.
[Gambar CDN: Select all file JSON]

Step 5 – Buka Tool CyberJSON

Buka link tool CyberJSON yang dibagikan secara eksklusif. Disarankan menggunakan browser utama (Chrome / Firefox).

Hindari browser Telegram atau Facebook karena sering bermasalah.
[Gambar CDN: link tool cyberjson]

Step 6 – Pilih File JSON

Klik Drop / Pilih File JSON, lalu cari folder kerja yang berisi file JSON.

[Gambar CDN: menu utama tool]

Jika hanya muncul recent file: buka menu garis tiga → File Manager / Memori Internal.

[Gambar CDN: cari file JSON]
[Gambar CDN: Pilih file JSON dari file manager]

Step 7 – Pilih File yang Akan Diedit / Translate

Umumnya file yang bisa diterjemahkan:

  • CommonEvents.json
  • MapXXX.json

Walaupun begitu, wajib mencoba semua file JSON karena teks kadang tersembunyi di file lain.

[Gambar CDN: Daftar file JSON di tool]

Step 8 – Edit Teks dengan Aman

Edit setiap kotak teks yang muncul.

JANGAN mengubah atau menghapus kode berikut:
  • \c[x]
  • \n
  • \
  • Tanda petik "
  • String ("xxx")
  • Kurung ( ) dan { }
Pastikan kode dan karakter sama persis seperti preview di atas kotak teks.
[Gambar CDN: Editor teks & indikator scroll]

Scroll hingga paling bawah sampai indikator persentase menunjukkan proses selesai.

[Gambar CDN: Editor teks & indikator scroll]

Step 9 – Simpan, Kembalikan File, dan Testing

  • Klik Save di pojok kanan atas
  • Buka folder Download
  • Rename file, hapus akhiran -id
  • Pindahkan kembali ke www/data
  • Jalankan game di emulator untuk testing
[Gambar CDN: Save & file hasil download]

⚠️ Peringatan

  1. Dilarang membagikan link tool ke publik. Khusus member Telegram.
  2. Dilarang membagikan hasil translate ke publik (kalau ingin di publikasikan bisa lewat jalur website admin, nama dan akun facebook akan di cantumkan di detail game,tinggal siapkan link vip dan link gratisnya. admin akan segerah mempublikasikan secara gratis).
  3. Hormati admin walau tidak berdonasi. tapi dengan mematuhi rules.
  4. Pelanggaran = blacklist + link tool diubah.
Benefit yang bisa didapat:

  4. Fix error hasil translate gratis
  5. Mendapat trik curang translate terbaru
  6. KALIAN BISA MEMILIH 1 ATAU SEMUA BENEFIT
